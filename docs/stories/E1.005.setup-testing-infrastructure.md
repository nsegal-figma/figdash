# Story E1.005: Setup Testing Infrastructure

## Status
Draft

## Story
As a frontend engineer I want to comprehensive testing infrastructure for chart components So that we can confidently ship high-quality visualizations with automated test coverage

## Description
Configure Jest, React Testing Library, visual regression testing (Chromatic), and accessibility testing (@axe-core/react) for the new visualization system. Create helper utilities and example tests that the team can follow.

## Acceptance Criteria
1. Jest configured for chart component testing
2. React Testing Library installed and configured
3. Mock utilities for canvas/SVG rendering
4. Visual regression testing setup (Chromatic or Percy)
5. Accessibility testing library installed (@axe-core/react, jest-axe)
6. Sample test written for a primitive component
7. Test coverage reporting configured (>85% target)
8. CI/CD pipeline updated to run viz tests
9. Test utilities documented in README
